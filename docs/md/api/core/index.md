# Core API Reference

Core FoundryVTT REST API Relay endpoints.

## Available Endpoint Groups

- [Roll](/api/core/roll/) - Dice rolling and roll history endpoints.
- [Entity](/api/core/entity/) - Entity creation, update, deletion, and item transfer endpoints.
- [Search](/api/core/search/) - Search endpoints for Foundry content.
- [Encounter](/api/core/encounter/) - Combat encounter management endpoints.
- [Structure](/api/core/structure/) - World structure, folder, and compendium endpoints.
- [Macro](/api/core/macro/) - Macro lookup and execution endpoints.
- [Sheet](/api/core/sheet/) - Actor sheet rendering endpoints.
- [Utility](/api/core/utility/) - Miscellaneous utility endpoints.
- [Session](/api/core/session/) - Headless session lifecycle endpoints.
- [File System](/api/core/fileSystem/) - File upload, download, and filesystem browsing endpoints.
